# tapog.ai Privacy Policy

Last updated: February 25, 2026

## Overview
`tapog.ai` is a Chrome extension that groups tabs by domain or category.

## Data the extension processes
The extension reads tab metadata in the current Chrome window:
- Tab URL
- Tab title

It also stores user preferences:
- Trigger key settings
- Grouping mode settings
- Custom Group Guard rules
- Custom AI category rules

## How data is used
Data is used only to perform tab grouping on the user's device.

## Data sharing and transfer
- No tab data is sold.
- No tab data is shared with third parties.
- No tab data is transmitted to external servers by this extension.

## Storage
Preferences are saved via `chrome.storage.sync` to support cross-device sync in the user's Chrome profile.

## Security
The extension does not implement remote code execution and does not fetch external scripts.

## Changes
This policy may be updated in future releases. The latest version will be published in this repository.
