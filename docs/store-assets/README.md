# Store Assets (tapog.ai)

This folder contains Chrome Web Store submission assets for `tapog.ai`.

Files:

- `screenshot-1.html`: static popup preview source for screenshot rendering
- `screenshot-1.png`: rendered Chrome Web Store screenshot
- `small-promo-tile.png`: `440x280` promo tile
- `marquee-promo-tile.png`: `1400x560` marquee tile
