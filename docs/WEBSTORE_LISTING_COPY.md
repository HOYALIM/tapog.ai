## Korean Copy

### Name
`tapog.ai`

### Short Description (option A)
AI 또는 도메인 기준으로 탭을 한 번에 정리하는 가장 빠른 방법.

### Short Description (option B)
복잡한 탭 정리, 이제 한 번의 클릭으로. Group by AI / Domain.

### Full Description
탭이 많아질수록 브라우징은 느려집니다.
`tapog.ai`는 탭 정리를 **한 번의 액션**으로 끝냅니다.

- **Group by AI**: 탭의 문맥을 기준으로 비슷한 카테고리끼리 묶습니다.
- **Group by Domain**: 도메인 기준으로 정확하고 예측 가능한 그룹을 만듭니다.
- **Group Guard**: 내가 지정한 URL 패턴은 항상 원하는 그룹으로 고정합니다.
- **Custom Trigger**: 원하는 키 조합 + 클릭으로 즉시 실행합니다.

개인 생산성을 위해 만든, 빠르고 미니멀한 탭 그룹 워크플로우.

### Single-line Pitch
탭 정리 시간을 줄이고, 집중 시간을 늘리는 익스텐션.

## English Copy

### Name
`tapog.ai`

### Short Description (option A)
Instantly organize tabs by AI or domain with one click.

### Short Description (option B)
From tab chaos to clean groups in one action.

### Full Description
When tabs pile up, focus drops.
`tapog.ai` gives you a fast, keyboard-friendly way to restore order.

- **Group by AI**: cluster tabs into meaningful categories.
- **Group by Domain**: deterministic grouping by registrable domains.
- **Group Guard**: force matching URLs into your predefined groups.
- **Custom Trigger**: launch grouping with your own key + click combo.

Built for people who work with lots of tabs and want instant structure.

### Single-line Pitch
Less tab chaos, more focus.
